# 🥔 The Potato Farm - Version 1.0
"The Potato Farm" is a farmming simulator where you can plant potatos in your farm, harvest and sell them in market.
It is a text-based game runs on Python 
## 🧪 Features
- Farm: Plant your own Potato! They grow even when you are offline!
- Market: A place where you can sell your potatos with varied price
- Shop: Use your money and buy seeds, extra slots
- Logbook: You can check your status and learn about different potatos
## ⚙️ Requirement
Before executing the game, you would need to have Python installed in your computer first.
- 📥 [Download Python from the Official Website](https://python.org)
- 💡 *Windows Users: Make sure to check the box that says **"Add Python to PATH"** during installation, or your terminal won't recognize the `python` command.*
## 📕 How to play
Very Simple!
use keyboard to input number(e.g. 1, 2, 3) and press "Enter" to confirm action<br>
Demo Video: https://youtu.be/MP7ziEfAOCg
## 📩 Installation
1. Download the zip file from github
2. Unzip the contents into folder
3. Open your terminal and change into this project folder:
   ```bash
   cd path/to/your/folder
   ```
4. Run the initiation file if it's your 1st time to play the game:
- **Windows:** `python init.py`
- **Mac / Linux:** `python3 init.py`
5. Run the main application file:
- **Windows:** `python main.py`
- **Mac / Linux:** `python3 main.py`
